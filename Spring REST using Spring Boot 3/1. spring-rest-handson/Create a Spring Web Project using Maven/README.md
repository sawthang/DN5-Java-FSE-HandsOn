# Spring Learn - Hands-on 1

## Objective

Create a Spring Boot web project using Maven.

## Technologies

-   Java 17
-   Spring Boot 3
-   Maven
-   Spring Web
-   Spring Boot DevTools

## Project Details

-   Group: `com.cognizant`
-   Artifact: `spring-learn`

## How to Run

1.  Open the project in IntelliJ IDEA or Eclipse.
2.  Update Maven dependencies.
3.  Run `SpringLearnApplication.java`.
4.  Verify that the application starts successfully on the default port.
