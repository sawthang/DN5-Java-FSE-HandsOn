# Spring REST Hands-on - Load Country from Spring Configuration XML

## Objective

Load a `Country` bean from a Spring XML configuration file and display
its details using the Spring IoC container.

## Technologies

-   Java 17
-   Spring Boot 3
-   Spring Core
-   Maven

## Features

-   XML-based bean configuration
-   `Country` bean with properties
-   Logging using SLF4J
-   Bean retrieval using `ApplicationContext`

## Files

-   `Country.java`
-   `SpringLearnApplication.java`
-   `country.xml`
-   `application.properties`
-   `pom.xml`

## How to Run

1.  Open the project in IntelliJ IDEA or Eclipse.
2.  Update Maven dependencies.
3.  Run `SpringLearnApplication.java`.
4.  Verify the country details in the console logs.
