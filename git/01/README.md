# GitDemo

## Objective
This repository was created as part of a Git Hands-on exercise to demonstrate basic Git operations.

## Tasks Performed
- Configured Git username and email
- Initialized a local Git repository
- Created a text file (`welcome.txt`)
- Checked repository status using `git status`
- Added the file to the staging area using `git add`
- Committed the changes using `git commit`
- Connected the local repository to GitHub
- Pushed the repository to GitHub

## Repository Structure

```
GitDemo/
├── README.md
└── welcome.txt
```

## Git Commands Used

```bash
git init
git status
git add welcome.txt
git commit -m "Added welcome.txt"
git remote add origin <repository-url>
git branch -M main
git push -u origin main
```
