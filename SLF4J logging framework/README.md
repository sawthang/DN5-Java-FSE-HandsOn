# SLF4J Logging Exercise

## Overview

This project demonstrates how to use **SLF4J** with **Logback** for logging messages in a Java application.

## Technologies Used

- Java
- Maven
- SLF4J
- Logback

## Features

- Logs an ERROR message.
- Logs a WARN message.
- Uses Logback as the logging implementation.

## Project Structure

```
src
└── main
    └── java
        └── LoggingExample.java
```

## How to Run

1. Clone the repository.
2. Open it in IntelliJ IDEA.
3. Reload Maven dependencies.
4. Run `LoggingExample.java`.

## Expected Output

```
ERROR LoggingExample -- This is an error message
WARN  LoggingExample -- This is a warning message
```

## Learning Outcome

- Understand SLF4J logging.
- Use Logback as a logging framework.
- Log messages at different severity levels.
