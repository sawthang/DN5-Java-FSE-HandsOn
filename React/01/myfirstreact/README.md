# myfirstreact

Hands-on lab exercise: create a React app that displays **"welcome to the first session of React"** as a heading.

## Setup steps (from the lab)

1. Install Node.js and NPM: https://nodejs.org/en/download/
2. Install Create React App globally:
   ```
   npm install -g create-react-app
   ```
3. Create the app:
   ```
   npx create-react-app myfirstreact
   ```
4. Move into the project folder:
   ```
   cd myfirstreact
   ```
5. Open the folder in Visual Studio Code.
6. Open `src/App.js`, remove its content, and replace it with the code in this repo's `src/App.js`.
7. Start the app:
   ```
   npm start
   ```
8. Open a browser and go to `http://localhost:3000` to see the heading.
