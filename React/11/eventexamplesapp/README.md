# eventexamplesapp

React hands-on lab: event handling.

## Covers

- Increment / Decrement counter (Increment also calls a second method that says hello with a static message)
- "Say Welcome" button that invokes a handler with `"welcome"` as an argument
- A button wired to a synthetic event (`OnPress`) that displays "I was clicked"
- A `CurrencyConvertor` component that converts Indian Rupees to Euros when the Convert button is clicked

## Run locally

```bash
npm install
npm start
```

Then open http://localhost:3000.
