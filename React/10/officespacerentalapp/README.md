# Office Space Rental App

A hands-on React JSX lab: `officespacerentalapp`.

## What this covers
- JSX syntax to create React elements
- Rendering an image via a JSX attribute
- An object (`office`) displaying Name, Rent, and Address
- A list of objects (`officeList`) rendered with `.map()`
- Inline CSS in JSX: rent text is **red** if below 60000, **green** if 60000 or above

## Run it

```bash
npm install
npm start
```

Then open http://localhost:3000.
